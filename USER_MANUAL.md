# 戦国戦況図タイムラインエディタ 操作説明書

## このアプリでできること

戦国時代の合戦図、進軍図、包囲図などを、時間の流れに沿って編集するためのツールです。

地図画像を背景に置き、その上にコマ、城・拠点、線、矢印、ラベルを配置できます。タイムライン上にキーフレームを打つことで、コマの移動、向き、サイズ、カメラ位置、カメラ倍率などを時間に合わせて変化させられます。完成した戦況図は PNG、JPEG、連番画像、MP4 として書き出せます。

## 画面構成

- 上部ツールバー: タイトル入力、編集ツール、ファイル操作、書き出し、キャンバス操作を行います。
- 左サイドバー: 陣営、コマ、拠点、アセットを管理します。
- 中央キャンバス: 戦況図を直接編集します。
- 右インスペクター: 選択中の要素の詳細設定を編集します。
- 下部タイムライン: 現在時間、キーフレーム、再生、補間を操作します。

## 基本操作

- 選択: 上部の「選択」ツールで、コマ・拠点・線・矢印・ラベルなどをクリックします。
- 複数選択: 何もない場所からドラッグして範囲選択します。選択した複数要素はまとめて移動できます。
- 移動: 選択した要素をドラッグします。
- パン: `Space` を押しながらドラッグします。
- ズーム: マウスホイールを使います。
- 表示位置リセット: 上部のリセットボタンを押します。
- フルスクリーン: ツールバー右端のフルスクリーンボタンで切り替えます。
- 削除: 要素を選択して `Delete` キーを押します。
- 元に戻す / やり直し: 上部ボタン、または `Ctrl + Z` / `Ctrl + Y` を使います。
- JSON保存: `Ctrl + S`、または「ファイル」メニューの「JSONを保存」を使います。

選択中のコマ・拠点・線・矢印・ラベルには、白黒の動く選択枠が表示されます。

## 領域選択

領域選択は、複数の要素をまとめて選択・移動したい時に使います。

### 領域で選択する

1. 上部の「選択」ツールを選びます。
2. キャンバス上の何もない場所から左ドラッグします。
3. 表示された四角い領域で、選びたい要素を囲みます。
4. マウスを離すと、領域にかかった要素がまとめて選択されます。

領域選択の対象は、コマ、城・拠点、線、矢印、ラベルです。地図画像、イベント、書き出しカメラは領域選択の対象にはなりません。

線や矢印は、領域が線の一部にかかっていれば選択されます。ラベルは、現在時間で表示されているものだけが対象になります。

### 選択した領域をまとめて動かす

複数の要素が選択されると、全体を囲む点線の領域枠が表示されます。この領域枠をドラッグすると、選択中の要素をまとめて移動できます。

移動結果は、現在時間の状態として反映されます。コマ、線、矢印は現在時間のキーフレームに更新され、城・拠点とラベルはその要素の位置が更新されます。ロックされている要素は、領域内に含まれていても移動しません。

### 領域選択を解除する

別の要素を単独でクリックすると、領域選択は解除されます。何もない場所をほとんど動かさずにクリックした場合も、選択は解除されます。

`Space` を押しながらドラッグするとパン操作になるため、領域選択は始まりません。

## タイトルとファイル操作

上部左側のタイトル欄は直接編集できます。このタイトルは JSON を保存する時のファイル名に使われます。

上部の「ファイル」メニューから以下を操作できます。

- 新規作成: すべての要素を削除し、空のプロジェクトを作ります。
- JSONを読み込む: 保存済み JSON を読み込みます。
- JSONを保存: 現在のプロジェクトを JSON として保存します。

地図画像、コマ画像、拠点画像は Data URL として JSON 内に保存されます。画像を多く含むプロジェクトは JSON ファイルが大きくなることがあります。

## 自動保存

編集内容は IndexedDB に自動保存されます。ページを更新しても、直前のプロジェクト、選択状態、現在ツール、描画中ポイント、キャンバスの表示位置と倍率が復元されます。

自動保存は同じブラウザ、同じURL元でだけ有効です。ブラウザのサイトデータを削除した場合や、別のブラウザで開いた場合は復元されません。共有や長期保存には JSON 保存を使ってください。

## 地図画像

地図画像は背景として扱います。通常の選択ツールではクリックできないため、コマや拠点の選択を邪魔しません。

### 地図画像を読み込む

1. 上部の「キャンバス」メニューを開きます。
2. 「地図画像を読み込む」を押します。
3. 画像ファイルを選びます。

読み込み後は自動で「地図画像編集」モードになります。

### 地図画像を編集する

1. 上部の「キャンバス」メニューを開きます。
2. 「地図画像編集」を押します。
3. 地図画像をドラッグして位置を調整します。
4. 右下の丸ハンドルをドラッグしてサイズを調整します。

地図画像のアスペクト比は固定です。幅と高さを別々に変えるのではなく、全体を拡大縮小します。

右インスペクターでは、地図画像の `X`、`Y`、`サイズ` を数値で調整できます。

## 陣営

左サイドバーの「陣営」タブで陣営を管理します。

- 陣営追加: 新しい陣営を追加します。
- 陣営選択: 陣営をクリックすると右インスペクターで編集できます。
- 色変更: 陣営行の色ボタンから色を変更します。
- 削除: 陣営行の削除ボタンで削除します。

陣営の色は、コマや拠点の枠・背景色に使われます。

## コマ

上部の「コマ」ツールを選ぶと、キャンバス上のマウス位置に半透明の配置プレビューが表示されます。クリックした位置にコマが配置されます。

新規コマの形状は、デフォルトで五角形です。必要に応じて右インスペクターで四角形に変更できます。

右インスペクターでは主に以下を編集できます。

- 名前、陣営
- ロック
- 形状: 四角形 / 五角形
- サイズ
- 名前の文字サイズ
- 名前表示、文字色、名前背景
- 画像アップロード / 画像削除
- 表示開始 / 表示終了
- 移動ルート
- 現在時間のキーフレーム

コマの名前はコマの下に表示されます。コマを回転した場合も、コマの表示部の下側に追従します。

### 五角形のコマ

五角形のコマは、上部が突き出た形で向きを表します。

コマを選択すると、周囲に回転用の丸ハンドルが表示されます。この丸をドラッグすると、コマの向きを自由に回転できます。回転角度は現在時間のキーフレームに保存されます。

画像付きの五角形コマでは、外側の五角形だけが回転し、中の画像は回転しません。

### コマ画像

「画像をアップロード」でコマに画像を設定できます。画像を設定している場合は「画像を削除」ボタンで解除できます。

透過 PNG の背景は陣営色で塗られます。画像は五角形の枠内に収まるように表示されます。

### コマのアセット

「アセットとして登録」を押すと、画像がないコマも含めて、名前、サイズ、形状、向き、陣営、名前表示設定などをアセットとして保存できます。名前が空欄のコマも、その状態のままアセット化できます。

アセットから配置する場合は、左サイドバーの「コマ」タブで「アセット」を押し、使いたいアセットを選びます。選択したアセットはすぐ配置されず、キャンバス上に半透明の配置プレビューが表示されます。クリックした位置に配置されます。

登録済みアセットは、アセット一覧の削除ボタンで削除できます。アセットから配置したコマを編集した場合は、右インスペクターの「編集後を新規アセット登録」で、現在の見た目を新しいアセットとして保存できます。

### 移動ルート

コマには線または矢印を移動ルートとして割り当てられます。

- ルートを追加: 使用する線・矢印、始点到達時間、終点到達時間、方向を指定します。
- ルート確認: ルート対象の線・矢印を確認表示します。
- すべて解除: コマに設定されたルートを解除します。

ルート移動の前後に通常の位置キーフレームがある場合は、ルート開始点・終了点につながるように補間されます。

## 城・拠点

上部の「城」ツールを選ぶと、キャンバス上のマウス位置に半透明の配置プレビューが表示されます。クリックした位置に城・拠点が配置されます。

拠点にはデフォルトの城画像が表示されます。カスタム画像をアップロードした場合は、「画像を削除」でカスタム画像だけを解除できます。

右インスペクターでは主に以下を編集できます。

- 名前
- 現在時間の陣営
- サイズ
- 名前の文字サイズ
- 名前表示、文字色、名前背景
- X / Y
- ロック
- 陣営キーフレーム
- 画像アップロード / 画像削除
- アセット登録（カスタム画像付き拠点）

カスタム画像付きの拠点はアセット登録できます。登録した拠点アセットは、左サイドバーの「拠点」タブで「アセット」を押すと表示されます。アセット一覧のアイコン背景には陣営色が反映されます。

## 線と矢印

上部の「線」または「矢印」ツールを選び、キャンバス上をクリックして点を追加します。

- `Enter`: 描画を確定します。
- `Esc`: 描画をキャンセルします。

右インスペクターでは、名前、色、太さ、透明度、点線、曲線、種類、キーフレームなどを編集できます。

線や矢印は、コマの移動ルートとして使うこともできます。選択中も元の線色はそのまま残り、周囲に選択枠が表示されます。

## ラベル

上部の「ラベル」ツールを選ぶと、キャンバス上のマウス位置に半透明の配置プレビューが表示されます。クリックした位置にラベルが配置されます。

ラベル配置後は、右インスペクターのテキスト入力欄に自動でフォーカスします。そのまま文字を入力して編集できます。

右インスペクターでは以下を編集できます。

- テキスト
- 表示開始 / 表示終了
- X / Y
- 文字サイズ
- 文字色
- 背景色
- 枠線色
- 透明度
- ロック
- メモ

## タイムラインとキーフレーム

下部タイムラインで現在時間を変更します。

- キーフレーム追加: 現在時間にタイムライン上のフレームを追加します。
- キーフレーム削除: 選択中のタイムラインフレームを削除します。
- 前 / 次: 前後のキーフレームへ移動します。
- 再生: 時間を進めて動きを確認します。
- ループ: 再生が最後まで進んだ後、最初に戻します。
- 補間: `linear` の場合、前後のキーフレーム間をなめらかに補間します。
- 速度: 再生速度を調整します。

タイムライン上のキーフレームをクリックすると、右インスペクターにその時間にキーを持つコマ、拠点、線、矢印、カメラの一覧が表示されます。一覧から対象を選択できます。

新しいキーフレームをそれまでの最終時間より後に追加した場合、それまで最終時間まで表示されていた要素の表示終了は、新しい最終時間まで自動で延長されます。

空のキーフレームは、コマのキーフレームを削除した時だけ自動整理されます。通常のキーフレーム追加やカメラ操作では自動削除されません。

## コマのキーフレーム

コマを選択すると、右インスペクターの「現在時間のキーフレーム」から現在時間にキーを追加・更新できます。

コマの位置、向き、サイズ、状態、陣営、確度などが時間ごとに保存されます。キャンバス上でコマをドラッグした場合も、現在時間のキーフレームが作成・更新されます。

コマのキーフレームをすべて削除しても、コマ自体は消えません。現在位置の情報を持ったコマとして残ります。

## カメラと書き出し範囲

書き出しは、キャンバス全体ではなく「書き出しカメラ」の範囲を使います。

### カメラを選択する

1. 上部の「キャンバス」メニューを開きます。
2. 「書き出しカメラ」を押します。

カメラ範囲の左上付近にあるカメラマークをクリックしても選択できます。カメラ範囲内をクリックしてもカメラは選択されないため、範囲内のコマや拠点を選択できます。

### カメラを動かす

カメラ選択中は、以下の方法でカメラ範囲を移動できます。

- カメラマークをドラッグする。
- カメラ枠線をドラッグする。

カメラ枠の内側をドラッグしても、カメラは移動しません。枠線だけが移動操作の対象です。

### カメラインスペクター

カメラを選択すると、右インスペクターで以下を設定できます。

- 出力W(px): 書き出し画像・動画の横ピクセル数
- 出力H(px): 書き出し画像・動画の縦ピクセル数
- 倍率: カメラの描画範囲の広さ
- X / Y: 現在時間のカメラ位置
- 現在の位置と倍率をキー登録

倍率を `2` にすると、出力サイズはそのままで、カメラの描画範囲が幅・高さとも 2 倍になります。広い範囲を同じ出力サイズに収めるため、書き出し結果は引いた見た目になります。

カメラの位置と倍率はキーフレーム化できます。補間モードが `linear` の場合、カメラ位置と倍率もなめらかに変化します。

## 書き出し

上部の「書き出し」メニューから操作します。

- 現在表示をPNG: 現在時間のカメラ範囲を PNG として出力します。
- 現在表示をJPEG: 現在時間のカメラ範囲を JPEG として出力します。
- PNG連番: タイムライン全体を連番 PNG の ZIP として出力します。
- JPEG連番: タイムライン全体を連番 JPEG の ZIP として出力します。
- MP4: タイムライン全体を MP4 動画として出力します。
- FPS: 連番画像 / MP4 のフレームレートを指定します。

MP4 書き出しはブラウザの対応状況に依存します。対応していない場合は PNG 連番または JPEG 連番を書き出し、動画編集ソフトで読み込んでください。

## 作業の流れ

1. キャンバスメニューから地図画像を読み込みます。
2. 地図画像編集で位置とサイズを調整します。
3. 陣営を設定します。
4. コマ、拠点、線、矢印、ラベルを配置します。
5. 必要なコマやカスタム画像付き拠点をアセット登録します。
6. タイムラインにキーフレームを追加します。
7. 時間を移動しながら、コマ位置、向き、サイズ、カメラ位置、倍率などを調整します。
8. 再生して動きを確認します。
9. 書き出しカメラの範囲を確認します。
10. PNG、JPEG、連番画像、MP4 のいずれかで書き出します。
11. 共有やバックアップ用に JSON を保存します。

## よくある注意点

- 自動保存はページ更新対策です。別環境へ渡す場合や長期保存には JSON 保存を使ってください。
- 地図画像は背景扱いです。通常はクリックできません。編集する時だけ「キャンバス」メニューの「地図画像編集」を使います。
- カメラの出力W/Hは書き出し解像度です。見える範囲を広げたい場合は倍率を上げます。
- 連番画像 / MP4 はカメラ範囲だけを書き出します。
- コマや拠点に画像を設定しても、削除ボタンで画像だけを解除できます。コマや拠点自体は削除されません。
- 画像付きプロジェクトは JSON ファイルや自動保存データが大きくなります。
- アセットを削除しても、すでに配置済みのコマや拠点は削除されません。
