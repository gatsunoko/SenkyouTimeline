# 戦国戦況図タイムラインエディタ 操作説明書

## このアプリでできること

戦国時代の合戦図・進軍図・包囲図などを、時間の流れに沿って編集するためのツールです。

地図画像を背景に置き、その上にコマ、城、線、矢印、ラベル、イベントを配置できます。タイムライン上にキーフレームを打つことで、コマの移動、向き、サイズ、カメラ位置などを時間に合わせて変化させられます。完成した戦況図は PNG、連番 PNG、MP4 として書き出せます。

## 画面構成

- 上部ツールバー: 編集ツール、ファイル操作、書き出し、キャンバス設定を操作します。
- 左サイドバー: 陣営、コマ、城、登録済みアセットを管理します。
- 中央キャンバス: 戦況図を直接編集します。
- 右インスペクター: 選択中の要素の詳細設定を編集します。
- 下部タイムライン: 現在時間、キーフレーム、再生、補間を操作します。

## 基本操作

- 選択: 上部の「選択」ツールで、コマ・城・線・矢印・ラベルなどをクリックします。
- 移動: 選択した要素をドラッグします。
- パン: `Space` を押しながらドラッグします。
- ズーム: マウスホイールを使います。
- 表示位置リセット: 上部のリセットボタンを押します。
- 削除: 要素を選択して `Delete` キーを押します。
- 元に戻す / やり直し: 上部ボタン、または `Ctrl + Z` / `Ctrl + Y` を使います。

## 領域選択

領域選択は、複数の要素をまとめて選択・移動したい時に使います。

### 領域で選択する

1. 上部の「選択」ツールを選びます。
2. キャンバス上の何もない場所から左ドラッグします。
3. 表示された四角い領域で、選びたい要素を囲みます。
4. マウスを離すと、領域にかかった要素がまとめて選択されます。

領域選択の対象は、コマ、城・拠点、線、矢印、ラベルです。地図画像、イベント、書き出しカメラは領域選択の対象にはなりません。

線や矢印は、領域が線の一部にかかっていれば選択されます。ラベルは、現在時間で表示されているものだけが対象になります。

### 選択した領域をまとめて動かす

複数の要素が選択されると、全体を囲む点線の領域枠が表示されます。この領域枠をドラッグすると、選択中の要素をまとめて移動できます。

移動結果は、現在時間の状態として反映されます。コマ、線、矢印は現在時間のキーフレームに更新され、城・拠点とラベルはその要素の位置が更新されます。ロックされている要素は、領域内に含まれていても移動しません。

### 領域選択を解除する

別の要素を単独でクリックすると、領域選択は解除されます。何もない場所をほとんど動かさずにクリックした場合も、選択は解除されます。

`Space` を押しながらドラッグするとパン操作になるため、領域選択は始まりません。

## プロジェクトの保存と読み込み

上部の「ファイル」メニューから操作します。

- JSONを保存: 現在のプロジェクトを JSON として保存します。
- JSONを読み込む: 保存済み JSON を読み込みます。

地図画像やコマ画像は Data URL として JSON 内に保存されます。画像を含むプロジェクトは JSON ファイルが大きくなることがあります。

## 地図画像

地図画像は背景として扱います。通常の選択ツールではクリックできないため、コマや城の選択を邪魔しません。

### 地図画像を読み込む

1. 上部の「キャンバス」メニューを開きます。
2. 「地図画像を読み込む」を押します。
3. 画像ファイルを選びます。

読み込み後は自動で「地図画像編集」モードになります。

### 地図画像を編集する

1. 上部の「キャンバス」メニューを開きます。
2. 「地図画像編集」を押します。
3. 地図画像をドラッグして位置を調整します。
4. 右下の丸ハンドルをドラッグしてサイズを調整します。

地図画像のアスペクト比は固定です。幅と高さを別々に変えることはできません。

右インスペクターでは、地図画像の `X`、`Y`、`サイズ` を数値で調整できます。

## 陣営

左サイドバーの陣営を選択すると、右インスペクターで名前、略称、色、種類、メモを編集できます。

陣営の色は、コマや城の枠・背景色に使われます。

## コマ

上部の「コマ」ツールでキャンバスをクリックすると、コマを追加できます。左サイドバーから既存コマを選択することもできます。

右インスペクターでは主に以下を編集できます。

- 名前、短縮名、陣営
- コマの種類、兵科、状態、確度
- サイズ
- 形状: 四角形 / 五角形
- 向き
- 名前表示、文字色、名前背景
- 画像アップロード / 画像削除
- アセット登録
- 移動経路

### 五角形のコマ

五角形のコマは、上部が突き出た形で向きを表します。

コマを選択すると、周囲に回転用の丸ハンドルが表示されます。この丸をドラッグすると、コマの向きを自由に回転できます。回転角度は現在時間のキーフレームに保存されます。

画像付きの五角形コマでは、外側の五角形だけが回転し、中の画像は回転しません。

### コマ画像

「画像をアップロード」でコマに画像を設定できます。画像を設定している場合は「画像を削除」ボタンで解除できます。

透過 PNG の背景は陣営色で塗られます。

### コマのアセット登録

「アセットとして登録」を押すと、画像がないコマも含めて、サイズ、形状、名前、表示設定などをアセットとして保存できます。

登録したアセットは左サイドバーから再利用できます。

## 城・拠点

上部の「城」ツールでキャンバスをクリックすると、城・拠点を追加できます。

右インスペクターでは主に以下を編集できます。

- 名前、陣営、状態、確度
- サイズ、名前表示、文字色、名前背景
- 画像アップロード / 画像削除
- 陣営キーフレーム
- アセット登録

城も画像なしでアセット登録できます。

## 線と矢印

上部の「線」または「矢印」ツールを選び、キャンバス上をクリックして点を追加します。

- `Enter`: 描画を確定します。
- `Esc`: 描画をキャンセルします。

右インスペクターでは、色、太さ、透明度、点線、曲線、種類、キーフレームなどを編集できます。

線や矢印は、コマの移動経路として使うこともできます。

## ラベルとイベント

ラベルは任意の注釈テキストとして使います。表示期間、文字サイズ、色、背景色、枠線色などを設定できます。

イベントは特定時刻に表示するマーカーです。戦闘開始、落城、撤退などのメモに使います。

## タイムラインとキーフレーム

下部タイムラインで現在時間を変更します。

- キーフレーム追加: 現在時間にタイムライン上のフレームを追加します。
- キーフレーム削除: 選択中のタイムラインフレームを削除します。
- 前後移動: 前後のキーフレームへ移動します。
- 再生: 時間を進めて確認します。
- 補間: `linear` の場合、前後のキーフレーム間をなめらかに補間します。

タイムライン上のキーフレームをクリックすると、右インスペクターにその時間にキーを持つコマ、城、線、矢印、カメラの一覧が表示されます。一覧から対象を選択できます。

空のキーフレームは、コマのキーフレームを削除した時だけ自動整理されます。通常のキーフレーム追加やカメラ操作では自動削除されません。

## コマのキーフレーム

コマを選択すると、右インスペクターの「現在時間のキーフレーム」から現在時間にキーを追加・更新できます。

コマの位置、向き、サイズ、状態、陣営、確度などが時間ごとに保存されます。

キャンバス上でコマをドラッグした場合も、現在時間のキーフレームが作成・更新されます。

## カメラと書き出し範囲

書き出しは、キャンバス全体ではなく「書き出しカメラ」の範囲を使います。

### カメラを選択する

1. 上部の「キャンバス」メニューを開きます。
2. 「書き出しカメラ」を押します。

カメラ範囲の左上付近にあるカメラマークをクリックしても選択できます。

### カメラを動かす

カメラ選択中は、以下の方法でカメラ範囲を移動できます。

- カメラマークをドラッグする。
- カメラ枠線をドラッグする。

カメラ枠の内側をドラッグしても、カメラは移動しません。枠線だけが移動操作の対象です。

### カメラインスペクター

カメラを選択すると、右インスペクターで以下を設定できます。

- 出力W(px): 書き出し画像・動画の横ピクセル数
- 出力H(px): 書き出し画像・動画の縦ピクセル数
- 倍率: カメラの描画範囲の広さ
- X / Y: 現在時間のカメラ位置
- 現在の位置と倍率をキー登録

倍率を `2` にすると、出力サイズはそのままで、カメラの描画範囲が幅・高さとも 2 倍になります。広い範囲を同じ出力サイズに収めるため、書き出し結果は引いた見た目になります。

カメラの位置と倍率はキーフレーム化できます。補間モードが `linear` の場合、カメラ位置と倍率もなめらかに変化します。

## 書き出し

上部の「書き出し」メニューから操作します。

- 現在表示をPNG: 現在時間のカメラ範囲を PNG として出力します。
- PNG連番: タイムライン全体を連番 PNG の ZIP として出力します。
- MP4: タイムライン全体を MP4 動画として出力します。
- FPS: 連番 PNG / MP4 のフレームレートを指定します。

MP4 書き出しはブラウザの対応状況に依存します。対応していない場合は PNG 連番を書き出し、動画編集ソフトで読み込んでください。

## 作業の流れ

1. キャンバスメニューから地図画像を読み込みます。
2. 地図画像編集で位置とサイズを調整します。
3. 陣営を設定します。
4. コマ、城、線、矢印、ラベルを配置します。
5. タイムラインにキーフレームを追加します。
6. 時間を移動しながら、コマ位置、向き、カメラ位置、倍率などを調整します。
7. 再生して動きを確認します。
8. 書き出しカメラの範囲を確認します。
9. PNG、PNG連番、MP4 のいずれかで書き出します。
10. JSON を保存して、後から編集できるようにします。

## よくある注意点

- 地図画像は背景扱いです。通常はクリックできません。編集する時だけ「キャンバス」メニューの「地図画像編集」を使います。
- カメラの出力W/Hは書き出し解像度です。見える範囲を広げたい場合は倍率を上げます。
- コマや城に画像を設定しても、削除ボタンで画像だけを解除できます。コマや城自体は削除されません。
- 画像付きプロジェクトは JSON ファイルが大きくなります。
- 連番 PNG / MP4 はカメラ範囲だけを書き出します。
