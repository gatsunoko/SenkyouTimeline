# 戦国戦況図タイムラインエディタ ユーザー向け説明書

## 概要

このアプリは、戦国時代の戦況図、進軍図、包囲図、領国図などを時間の流れに沿って作成するための編集ツールです。

地図画像の上に、コマ、拠点、画像、領域、線、矢印、ラベルを配置できます。時間ごとにキーフレームを登録すると、コマの移動、向き、サイズ、領域の形、線や矢印の形、ラベルや画像の位置、カメラ位置と倍率を変化させられます。

完成した内容は、プロジェクト JSON として保存できるほか、書き出しカメラの範囲を PNG、JPEG、連番画像、MP4 として書き出せます。

## 画面構成

- 上部ツールバー: タイトル、配置ツール、履歴、ファイル、書き出し、キャンバス、フルスクリーンを操作します。
- 左サイドバー: 陣営、コマ、拠点、画像、領域、線、矢印、ラベルの一覧とアセットを管理します。
- 中央キャンバス: 地図とオブジェクトを直接編集します。
- 右インスペクター: 選択中の項目の詳細設定を編集します。
- 下部タイムライン: 現在時間、再生、速度、終了秒数を操作します。

## 基本操作

- 選択: 上部ツールバーの選択状態で、キャンバス上のオブジェクトをクリックします。
- 複数選択: 何もない場所からドラッグして範囲選択します。
- 移動: 選択したオブジェクトをドラッグします。複数選択中はまとめて移動できます。
- パン: `Space` を押しながらキャンバスをドラッグします。
- ズーム: マウスホイールで拡大縮小します。
- 表示位置リセット: ツールバーのリセットボタンを押します。
- フルスクリーン: ツールバー右端のボタンで切り替えます。
- 削除: オブジェクトを選択して `Delete` を押します。
- 描画確定: 線、矢印、領域の点を打っている時に `Enter` を押します。
- キャンセル: 描画中に `Esc` を押します。
- 元に戻す / やり直し: ツールバーのボタン、または `Ctrl + Z` / `Ctrl + Y` を使います。
- JSON保存: `Ctrl + S`、またはファイルメニューの「JSONを保存」を使います。

選択中のオブジェクトには、白と黒が交互に動く選択表示が出ます。線や矢印は元の色の上を塗りつぶさず、周囲に選択表示が出ます。

## ロック

コマ、拠点、画像、領域、線、矢印、ラベルはロックできます。

ロックすると、キャンバス上ではクリック選択、ドラッグ移動、範囲選択の対象になりません。背景や下にあるオブジェクトを編集したい時に便利です。ロック解除は左サイドバーのロックボタン、または右インスペクターから行います。

左サイドバーのロックボタンは、ロック中に赤い背景で表示されます。

## ファイルと自動保存

上部左の「タイトル」入力欄は、プロジェクト名です。JSONを保存した時のファイル名にも使われます。

ファイルメニューでは以下を操作できます。

- 新規作成: すべての要素を削除して空のプロジェクトに戻します。
- JSONを読み込む: 保存済みのプロジェクト JSON を読み込みます。
- JSONを保存: 現在のプロジェクトを JSON として保存します。

編集内容は IndexedDB に自動保存されます。ページを更新しても、直前のプロジェクト、選択状態、配置モード、描画中の点、キャンバス表示位置が復元されます。

自動保存は同じブラウザ、同じ URL 元でだけ有効です。長期保存、バックアップ、別環境への移動には JSON 保存を使ってください。

## 地図画像

地図画像は背景として扱われます。通常の選択モードではクリックできないため、コマや拠点の選択を邪魔しません。

地図画像を読み込むには、キャンバスメニューの「地図画像を読み込む」を使います。読み込むと自動的に「地図画像編集」モードになります。

地図画像を調整するには、キャンバスメニューの「地図画像編集」を選びます。このモード中だけ地図画像をドラッグできます。右下のハンドルをドラッグすると、アスペクト比を固定したまま拡大縮小できます。

右インスペクターでは、地図画像の X、Y、サイズを数値で調整できます。

## 書き出しカメラ

書き出しはキャンバス全体ではなく、書き出しカメラの範囲を使います。

キャンバスメニューの「書き出しカメラ」を押すか、カメラ範囲左上付近のカメラマークをクリックすると、カメラを選択できます。カメラ範囲の内側をクリックしてもカメラは選択されないため、範囲内のコマや拠点をそのまま選択できます。

カメラ選択中は、カメラマークまたはカメラ枠線をドラッグするとカメラ範囲を移動できます。枠線の内側をドラッグしてもカメラは移動しません。

カメラインスペクターでは以下を設定できます。

- 出力W(px): 書き出す画像や動画の横ピクセル数。
- 出力H(px): 書き出す画像や動画の縦ピクセル数。
- 倍率: カメラが見る範囲の倍率。
- X / Y: 現在時間のカメラ位置。
- 現在の位置と倍率をキー登録: 現在時間にカメラのキーフレームを追加または更新します。

倍率を大きくすると、同じ出力サイズの中により広い範囲が入ります。カメラ位置と倍率はキーフレーム化でき、時間に沿って移動できます。

## 書き出し

書き出しメニューでは以下を使えます。

- 現在表示をPNG: 現在時間のカメラ範囲を PNG で保存します。
- 現在表示をJPEG: 現在時間のカメラ範囲を JPEG で保存します。
- FPS: 連番画像と MP4 のフレームレートです。
- PNG連番: タイムライン全体を連番 PNG の ZIP として保存します。
- JPEG連番: タイムライン全体を連番 JPEG の ZIP として保存します。
- MP4: タイムライン全体を MP4 動画として保存します。

MP4 書き出しはブラウザの対応状況に依存します。うまく書き出せない場合は、PNG連番またはJPEG連番を書き出して動画編集ソフトで読み込んでください。

## タイムライン

下部タイムラインでは、現在時間をスライダーで移動できます。

- 再生 / 停止: タイムラインを再生または停止します。
- 速度: 再生速度を変更します。
- 終了秒数: プロジェクトの終了時刻を直接入力します。

現在の補間は線形補間です。オブジェクトに複数のキーフレームがある場合、前後のキーフレームの間をなめらかに補間します。

以前のようなタイムライン上のキーフレーム一覧や、全体のキーフレーム追加 / 削除ボタンはありません。キーフレームは各オブジェクトのインスペクターから追加、更新、削除します。

新しい終了秒数を、それまでの最終時刻より後に設定した場合、それまで最終時刻まで表示されていたオブジェクトは、新しい終了時刻まで表示されます。

## 左サイドバー

左サイドバーには、以下のタブがあります。

- 陣営
- コマ
- 拠点
- 画像
- 領域
- 線
- 矢印
- ラベル

コマ、拠点、画像は、配置済み一覧とアセット一覧を切り替えられます。アセットをクリックすると配置準備になり、キャンバス上に半透明のプレビューが表示されます。

各一覧には表示開始と表示終了が表示されます。現在時間に表示中のオブジェクトは白、表示時間外のオブジェクトは灰色で表示されます。キャンバスでオブジェクトを選択すると、左サイドバーもそのオブジェクトが見える位置まで自動スクロールします。

## 陣営とカメラ内陣営表示

陣営タブでは、陣営の追加、名前、色、メモ、削除を管理します。陣営色はコマ、拠点、領域、凡例表示に使われます。

陣営タブをクリックすると、右インスペクターに「陣営表示設定」が表示されます。

- カメラに陣営情報を表示: 書き出しカメラの左上に陣営一覧を表示します。
- 表示サイズ: カメラ内陣営表示の大きさを調整します。
- 表示する陣営: 陣営ごとに表示するかを切り替えます。
- 文字アウトライン色: 陣営名のアウトライン色を指定します。

陣営表示はカメラに追従します。キャンバスをズームしても、PNGや動画に書き出される実際のサイズに近い大きさで表示されます。

## コマ

上部ツールバーの「コマ」を選ぶと、キャンバス上のマウス位置に半透明のコマが表示されます。クリックするとその位置にコマを配置します。通常のコマ追加は1個配置すると選択モードに戻ります。

コマのデフォルト形状は五角形です。インスペクターで以下の形状に変更できます。

- 四角形
- 五角形
- 凸型

五角形と凸型は向きを持つコマです。コマを選択すると回転用の丸ハンドルが表示され、ドラッグで自由に回転できます。回転角度もキーフレームに保存されます。画像付きのコマでは、外側の枠だけが回転し、内側の画像は回転しません。

コマの名前はコマの下に表示されます。コマを回転しても、表示されているコマ本体の下側に追従します。

コマインスペクターでは主に以下を編集できます。

- 名前
- 陣営
- ロック
- 形状
- サイズ
- 名前の文字サイズ
- 画像アイコンのアップロード / 削除
- 名前表示、太字、文字色、背景、アウトライン
- アセット登録
- 表示期間
- 移動ルート
- 現在時間のキーフレーム
- このコマのキーフレーム

「現在時間にキーフレーム追加/更新」を押すと、現在時間にコマの位置、サイズ、向きなどを保存します。コマのキーフレーム一覧をクリックすると、その時間へ移動できます。

コマのキーフレームをすべて削除しても、コマ自体は消えず、その場に残ります。

## コマアセット

コマは、画像がある場合もない場合もアセットとして登録できます。名前が空欄でも、その状態を含めて登録されます。

アセットには、名前、画像、サイズ、陣営、形状、向き、名前表示、文字サイズ、文字色、太字、背景、アウトラインなどが保存されます。

左サイドバーのコマタブで「アセット」を押し、登録済みアセットをクリックすると配置準備になります。コマアセットから配置する場合は、1個置いても選択モードに戻らず、連続して配置できます。

アセットから配置したコマを編集した後は、「編集後を新規アセット登録」で、編集後の状態を別アセットとして保存できます。

## コマの移動ルート

コマには、線または矢印を移動ルートとして割り当てられます。

移動ルートでは、使用する線・矢印、始点到達時間、終点到達時間、方向を設定します。複数のルートを追加することもできます。

「ルート確認」を押すと、ルート対象の線や矢印を確認できます。「通常は非表示（ルート確認で表示）」をオンにした線や矢印は、通常表示では隠し、ルート確認時だけ表示できます。

通常の位置キーフレームとルート移動が同じコマにある場合、ルート開始前はルートの開始点へ向かい、ルート終了後は次のキーフレームへ移動するように補間されます。

## 拠点

上部ツールバーの「城」を選ぶと、キャンバス上に半透明の拠点プレビューが表示されます。クリックすると拠点を配置します。配置後は選択モードに戻ります。

拠点にはデフォルトの城画像が使われます。必要に応じてカスタム画像をアップロードできます。カスタム画像は「画像を削除」で解除できます。

拠点インスペクターでは主に以下を編集できます。

- 名前
- 現在時間の陣営
- サイズ
- 名前の文字サイズ
- X / Y
- 名前表示、太字、文字色、背景、アウトライン
- ロック
- 陣営キーフレーム
- 画像アップロード / 削除
- アセット登録
- メモ

拠点は画像がなくてもアセット登録できます。アセットには、名前、画像、サイズ、陣営、名前表示設定などが保存されます。

## 画像オブジェクト

上部ツールバーの「画像」を選ぶと画像ファイルを選択します。選択後、キャンバス上に半透明の画像プレビューが表示され、クリックした位置に画像オブジェクトを配置します。

画像オブジェクトは、木、山、家紋、装飾、背景素材など、名前を表示しない画像素材を配置するための機能です。キーフレーム化できるのは座標です。

画像インスペクターでは以下を編集できます。

- 名前
- サイズ
- X / Y
- ロック
- 画像差し替え
- アセット登録
- 座標キーフレーム
- メモ
- 画像オブジェクト削除

画像オブジェクトをロックすると、キャンバス上ではクリック選択できません。背景素材として置いた画像を邪魔にしたくない場合に使います。解除は左サイドバーまたはインスペクターから行います。

画像オブジェクトもアセット登録できます。アセットから配置する時の半透明プレビューは、登録されているサイズで表示されます。

## 領域

上部ツールバーの「領域」を選び、キャンバス上をクリックして点を打ちます。3点以上打った状態で `Enter` を押すと領域が作成されます。`Esc` でキャンセルできます。

領域は国、勢力圏、支配範囲などを色分けして表示するための機能です。

領域インスペクターでは以下を編集できます。

- 名前
- 陣営
- 塗り色を陣営色に連動
- 塗り色
- 透明度
- 表示順
- 境界線の表示、色、幅
- 名前表示、太字
- ロック
- 表示期間
- 点の編集
- 領域キーフレーム
- メモ

領域は表示順が大きいほど上に表示されます。領域同士が重なった場合、上にある領域が優先され、下の領域の重なった部分は見えないように処理されます。

領域を選択すると、点をドラッグして形を変えられます。「現在の領域をキーフレームに追加」を押すと、現在時間の領域形状として保存されます。時間ごとに領域の形を変えることで、支配範囲の変化を表現できます。

領域選択中は、範囲選択で領域の点を2点まで選べます。2点を選ぶと、キャンバス上とインスペクターに「間に点を追加」操作が表示され、選んだ2点の中間に新しい点を追加できます。

## 線

上部ツールバーの「線」を選び、キャンバス上をクリックして点を打ちます。2点以上打った状態で `Enter` を押すと線が作成されます。`Esc` でキャンセルできます。

線インスペクターでは以下を編集できます。

- 名前
- 色
- 太さ
- 透明度
- 点線
- 線の形: 直線 / 曲線
- 通常は非表示（ルート確認で表示）
- 表示期間
- 現在時間の点
- 線キーフレーム
- メモ

線の点はドラッグで移動できます。2点を選ぶと、その間に点を追加できます。線はコマの移動ルートとして使えます。

## 矢印

上部ツールバーの「矢印」を選び、キャンバス上をクリックして点を打ちます。2点以上打った状態で `Enter` を押すと矢印が作成されます。`Esc` でキャンセルできます。

矢印インスペクターでは以下を編集できます。

- 名前
- 色
- 太さ
- 先端サイズ
- 透明度
- 点線
- 矢印の形: 直線 / 曲線
- 通常は非表示（ルート確認で表示）
- 始点から終点に沿って表示
- 表示にかける秒数
- 表示期間
- 現在時間の点
- 矢印キーフレーム
- メモ

「始点から終点に沿って表示」をオンにすると、表示開始時に矢印全体が一気に出るのではなく、始点から先端に向かって指定秒数かけて伸びます。曲線の場合も最終的な曲線に沿って伸びます。

矢印の点もドラッグで移動できます。2点を選ぶと、その間に点を追加できます。矢印はコマの移動ルートとしても使えます。

## ラベル

上部ツールバーの「ラベル」を選ぶと、キャンバス上に半透明のラベルプレビューが表示されます。クリックするとラベルを配置します。配置後は右インスペクターのテキスト入力欄にフォーカスが移り、そのまま編集できます。

ラベルインスペクターでは以下を編集できます。

- テキスト
- 表示期間
- X / Y
- 文字サイズ
- 文字を太字
- 文字色
- 背景色の表示 / 非表示
- 背景色
- 文字アウトラインの表示 / 非表示
- アウトライン色
- 枠線の表示 / 非表示
- 枠線色
- 透明度
- ロック
- 座標キーフレーム
- メモ

ラベルは位置をキーフレーム化できます。時間に合わせてテキスト位置を動かしたい場合は、現在時間を移動してから「現在の座標をキーに追加/更新」を押します。

## 表示期間

コマ、領域、線、矢印、ラベルなどには「表示開始」と「表示終了」があります。

この値はキーフレーム一覧から選ぶのではなく、秒数を直接入力します。キーフレームが存在しない秒数にも設定できます。

表示期間外のオブジェクトはキャンバス上に表示されません。左サイドバーでは、表示期間外のオブジェクト名が灰色になります。

## キーフレーム

キーフレームは各オブジェクトのインスペクターから管理します。

- コマ: 位置、サイズ、向きなど。
- 拠点: 陣営。
- 画像: 座標。
- 領域: 点の並び。
- 線: 点の並び。
- 矢印: 点の並び。
- ラベル: 座標。
- カメラ: 位置と倍率。

キャンバス上でドラッグした場合も、対象によって現在時間のキーフレームが作成または更新されます。

0秒のフレームと、存在する最後の秒数は、オブジェクトのキーがなくても残ります。不要なオブジェクト別キーフレームは、それぞれのインスペクターから削除します。

## 配置モードと描画中の注意

線、矢印、領域を新しく配置している間は、他のオブジェクトをドラッグできません。描画中の点や線には、背景色に埋もれないように白黒の補助表示が出ます。

上部ツールバーからコマ、城、画像、ラベルを配置した場合は、1個置いたあと選択モードに戻ります。コマアセットから配置する場合は、連続配置しやすいようにコマ配置モードが維持されます。

## 典型的な作業手順

1. キャンバスメニューから地図画像を読み込みます。
2. 地図画像編集で位置とサイズを合わせます。
3. 陣営を追加し、名前と色を設定します。
4. 必要ならカメラに陣営情報を表示する設定を行います。
5. コマ、拠点、画像、領域、線、矢印、ラベルを配置します。
6. よく使うコマ、拠点、画像をアセット登録します。
7. タイムラインの現在時間を移動しながら、各オブジェクトのキーフレームを登録します。
8. 書き出しカメラのサイズ、位置、倍率を調整します。
9. 再生して動きを確認します。
10. PNG、JPEG、連番画像、MP4 のいずれかで書き出します。
11. バックアップ用に JSON を保存します。

## 注意点

- 自動保存はページ更新対策です。完成データや共有用データは JSON として保存してください。
- 地図画像は通常クリックできません。位置やサイズを変える時だけ「地図画像編集」を使います。
- ロック中のオブジェクトはキャンバス上で選択できません。左サイドバーから選択して解除してください。
- 画像を多く含むプロジェクトは JSON と自動保存データが大きくなります。
- アセットを削除しても、すでに配置済みのオブジェクトは削除されません。
- カメラの出力W/Hは書き出し解像度です。見える範囲を広げたい時は倍率を上げます。
- オブジェクトの座標は地図に対する相対値です。カメラの X/Y はピクセル基準です。
- 書き出し結果は書き出しカメラの範囲だけです。キャンバス全体ではありません。
